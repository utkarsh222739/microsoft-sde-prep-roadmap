# Microsoft SDE Interview Roadmap · IDC Prep Guide

An experience-mapped roadmap to crack **Software Development Engineer** interviews at Microsoft's India Development Center (IDC) — from DSA fundamentals to distributed systems and behavioral rounds, with the best curated resource for each level.

🔗 **Live site:** https://utkarsh222739.github.io/microsoft-sde-prep-roadmap/

## What's inside

- **Interview loop overview** — OA → phone screen → onsite loop → "As-Appropriate" round → debrief.
- **SDE 1 (0–2 yrs)** — DSA patterns, CS fundamentals (OS, DBMS, Networking) and a study rhythm.
- **SDE 2 (2–5 yrs)** — Low-Level Design (SOLID, patterns), High-Level Design, and project ownership.
- **Senior / Principal (5+ yrs)** — Advanced distributed systems, design execution and leadership.
- **Behavioral (all levels)** — STAR method and the Growth Mindset signals Microsoft looks for.
